# yaopulife 选品运营工作流

## SOP执行流程

```
市场调研 → 选品筛选 → 样品采购 → AI生图 → 质检精修 → 上品执行 → 日常运营
```

## Step 2: 1688选品操作

**搜索关键词**：
- 针毡宠物肖像： https://www.1688.com/ - 搜索"针毡 宠物 肖像"
- 定制刺绣： https://www.1688.com/ - 搜索"定制 刺绣 挂圈"
- 月球灯： https://www.1688.com/ - 搜索"月球灯 3D打印 定制"
- 树脂工艺品： https://www.1688.com/ - 搜索"树脂 杯垫 海洋"

**筛选条件**：
- 交易等级：≥3星
- 复购率：≥15%
- 响应时效：≤2小时
- 支持来样定制

**记录信息**：
每款候选产品记录到 `PRODUCT-SELECTION-TRACKER.md`：
- 店铺名称
- 商品链接
- 进货价格
- 起订量
- 生产周期

---

## Step 3: 供应商评估

**评估5维度**（每项1-5分）：
1. 供货稳定性
2. 定制能力
3. 价格竞争力
4. 品质保障
5. 沟通效率

**达标标准**：
- 综合评分 ≥4.0/5
- 支持OEM/ODM
- 起订量 ≤50件
- 24h内回复

---

## Step 4: 样品采购

**样品采购流程**：
1. 从候选供应商采购1-2件样品
2. 记录采购信息到表格
3. 验收质量（材质/工艺/定制效果）
4. 通过后锁定主力供应商

**样品验收清单**：
- [ ] 材质与描述一致
- [ ] 定制区域效果清晰
- [ ] 包装适合国际运输
- [ ] 无异味/安全问题

---

## Step 5: AI生图（OpenClaw）

**生图类型**：
| 类型 | 数量 | 用途 |
|------|-----|------|
| 白底主图 | 1张 | 产品页首图 |
| 场景图 | 2-3张 | 生活方式展示 |
| 定制效果图 | 2张 | 展示定制效果 |
| 细节特写 | 1-2张 | 材质/工艺展示 |
| 社交媒体图 | 2-3张 | Instagram/Pinterest |

**生图Prompt模板**：
```
[产品类型] + [材质] + [定制元素] + [背景] + [风格] + [技术参数]
```

**质检标准**：
- 产品形态一致
- 定制区域清晰
- 无AI畸变
- 色彩准确
- 通过率 ≥80%

---

## Step 6: 上品执行

**上品清单**：
1. 产品标题：`Brand | Type - Customization | Benefit`
2. 产品描述：≥300词，自然融入关键词
3. 图片上传：5-8张，含Alt标签
4. 定制选项：配置定制器+价格
5. SEO元数据：Meta Title/Description/URL
6. 内链：2-3个相关产品+1篇博客

---

## Step 7: 订单履约

**SLA标准**：
| 环节 | 时效 |
|------|------|
| 订单确认 | 即时 |
| 定制内容审核 | 24h |
| 设计确认 | 48h |
| 客户确认等待 | 72h |
| 生产周期 | 5-18天 |
| 总履约 | ≤14天 |
